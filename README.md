# sentinel-ai
AI-driven digital asset protection prototype for sports media

## Overview
SentinelAI detects, analyzes, and predicts unauthorized usage of sports media across platforms. It uses content fingerprinting, AI-based reasoning, and propagation tracking to identify high-risk violations and prevent content misuse before it goes viral.

## Features
- Content fingerprinting and similarity detection
- AI-powered risk analysis using Gemini
- Propagation graph visualization
- Threat intelligence and alert system
- Predictive “anomaly-before-viral” detection

## Tech Stack
- HTML (embedded JavaScript + embedded CSS)
- D3.js (graph visualization)
- Google Gemini API (AI analysis)

## How to Run
Open `index.html` in a browser.
